## 📝 本地化内容类型

每个命令需要本地化的内容包括：

1. **命令描述** (describe)
2. **选项描述** (builder 中的 description)
3. **日志输出** (log.show, log.info 等)
4. **错误消息** (log.error)
5. **用户提示** (inquirer 消息)
6. **状态信息** (处理进度、结果统计等)

## 🎨 质量要求

- **一致性**: 相同概念使用相同翻译
- **准确性**: 技术术语准确无误
- **简洁性**: 消息简洁明了
- **友好性**: 用户友好的表达方式
